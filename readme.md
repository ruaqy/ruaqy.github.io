## 文件系统

 |文件夹			|含义								|
 |--------------|-----------------------------------|
 |ccs			|存放ccs文件							|
 |img			|存放运行时所需要的图片				|
 |js			|存放运行所用的js文件					|
 |pages			|存放一些界面文稿						|
 |pages/post	|存放能被显示的文稿(会被git记录)		|
 |pages/draft	|存放不会被显示的草稿(不会被git记录)	|
